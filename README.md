# Business Performance Dashboard

<img width="1033" height="649" alt="image" src="https://github.com/user-attachments/assets/18e4dd92-013d-4610-ba5c-c8beffae0c17" />

This project presents an interactive business performance dashboard built in Excel.
The dashboard provides a consolidated view of revenue, profit, margin, sales volume, and regional performance, enabling management-level decision-making.
The goal of this project was to design a structured reporting system that transforms raw transactional data into actionable insights.
